Das maisons parisienses às grifes milanesas, uma linguagem visual comum emerge nesta temporada: a celebração da estrutura depurada. Não é a sobriedade fria do minimalismo dos anos 90 — é algo mais rico, mais deliberado.

Os ombros pronunciados retornam como símbolo de autoridade. O tecido fala por si: lã de merino prensada, seda matelassê, couro vegano de textura quase escultural. A paleta se concentra nos neutros quentes — marfim, pedra, areia — com desvios pontuais para o preto-carvão que domina a noite.

O que diferencia este movimento é a atenção obsessiva ao caimento. Cada costura existe por uma razão. Cada dobra é calculada. É moda que respeita o corpo sem aprisioná-lo.
