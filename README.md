# MAISON — Portal Editorial de Moda

A premium fashion editorial website inspired by Vogue's visual identity, combining fashion trend coverage with film curation from the fashion universe.

## Key Technologies

- **TanStack Start** (React, file-based routing)
- **Content Collections** — markdown-based editorial content
- **Tailwind CSS v4** — utility-first styling
- **Google Fonts** — Cormorant Garamond + DM Sans for luxury editorial typography
- **Netlify** — hosting and deployment

## Features

- Full editorial homepage with hero, trending section, film curation, weekly feature, newsletter, and footer
- Content-driven architecture with markdown posts for fashion articles
- Sophisticated animations: marquee ticker, fade-up reveals, image zoom on hover
- Fully responsive (mobile + desktop)
- Premium neutral palette: warm white, off-black, taupe, sand

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
